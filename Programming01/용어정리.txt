문장: 명령어 하나
;(세미콜론): 문장이 끝났다는것을 알리기위한 기호
구문(or블록): {}(중괄호)로 묶여있는 문장들의 단위 or 지역변수의 생존범위

변수: 데이터를 가지고 있는것
-int: 정수
-float: 실수
-string: 문자열
함수: 필요할때 불러쓰는 구문
주석: 실행되지 않는 문장(알아보기 쉽게 필기하는 용도)
매개변수: 함수 외부에서 값을 받아와서 문장을 실행하기 위한 변수
리턴: 함수의 값을 호출
전역변수: 함수(구문) 밖에서도 사용 가능한 변수
지역변수: 함수(구문) 안에서만 사용 가능한 변수

class: 
public: 클래스 바깥에서 사용할 수 있도록 해주는 것
private: 클래스 안에서만 사용할 수 있도록 해주는 것

[디버깅]
F9: 브레이킹 포인트
F11: 변수의 값을 볼 수 있음
F12: 함수의 원본을 볼 수 있음

[게임시스템 기획]
퀘스트: 보상을 받기위한 의뢰
인벤토리: 소지하고있는 물품, 장비 등을 볼수있는 창
쿨타임: 재사용 가능하기까지의 남은 시간
